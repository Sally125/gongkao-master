

本项目基于 [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app) 脚手架初始化 [Next.js](https://nextjs.org) 框架项目，采用 App Router 目录结构开发。

---

## 📋 环境要求

- Node.js >= 18.17
- 包管理器：npm 

---

## 🚀 快速开始

### 1. 安装依赖（首次拉取代码需要执行）

```bash
npm install
```

### 2. 启动本地开发服务

```bash
npm run dev
```

在浏览器访问 [http://localhost:3000](http://localhost:3000) 预览页面。


---

## 📦 打包部署

### 构建生产环境代码

```bash
npm run build
```

### 本地运行生产构建产物

```bash
npm run start
```
